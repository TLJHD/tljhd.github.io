About the mod

The Longest Journey HD is a mod for one of the best adventure games ever made. It includes neural network upscaled backgrounds and sprites, redrawn user interface and a lot of other small improvements thanks to ResidualVM implementation of the game engine.

I wouldn't call it the final release, rather version 1.0. There is always room for improvement, because SISR algorithms are not perfect and the quality of the original graphics assets is not consistent.

Installation instructions

  • 1. Install The Longest Journey. If you don't own the game, you can buy it on GOG or Steam.
  • 2. Download the main mod and extract the contents of TLJHD.zip to the game installation directory. If asked, overwrite all existing files. Make sure that the path to the mod looks like this: The Longest Journey\mods\TLJHD\. You can also download upscaled in-game videos and extract them to the game directory as well.
  • 3. Download ResidualVM for your system and extract files to any folder. This is an essential step, because the original game engine doesn't support modified assets and overall have many issues on modern systems.
  • 4. Launch ResidualVM, click Add game... and choose the game (not the mod!) directory.
  • 5. In the Graphics tab select Override global graphic settings and Fullscreen mode. Click OK.
  • 6. Enable anti-aliasing for 3D models. Click Options... and select the level of Anti-aliasing in a drop-down list. From my experience 4x is more than enough for The Longest Journey.
  • 7. Restart ResidualVM.
  • 8. Click Start to launch the game. Ignore the popup message and click Start anyway.
  • 9. Enjoy the game!

ResidualVM supports keyboard shortcuts for different in-game actions.
You can view the full list on the official GitHub page.
